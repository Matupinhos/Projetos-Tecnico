Por: Matheus Bernardes

Nesse arquivo vou explicar algumas tags para usar na criação de sites em HTML.

HTML:

A tag HTML serve para começar o arquivo em HTML.

HEAD:

A tag head serve para providenciar as informações mais gerais do site. Exemplo: título e linguagem do teclado.

HEADER:

A tag header server pra criar um cabeçalho para o site.

BODY:

A tag body serve para criar "corpo" do site. É o lugar aonde a maior parte das informações do site vão estar.

SECTION:

A tag section é usada pra criar uma seção no corpo do texto. Você pode criar várias delas para organizar o texto.

DIV:

A tag div é usada para divir o codigo em partes Pode ser usada dentro de uma section, como tambem uma section pode ser usada na div.

FOOTER:

A tag footer cria um rodapé no texto.

H1 ate H7:

A tag h1 é usada para título, subtítulos, sub subtítulos, etc. O h1 é o título, h2 subtítulo, etc. Quanto maior o número, menor a fonte da letra vai ser

HR:

A tag hr cria uma linha visual no texto. Pode ser usada para organizar o texto.

P:

A tag p é usada pra criar um parágrafo. Aonde você pode escrever as informações importantes.É usada após os títulos e subtítulos

UL:

A tag ul é usada para criar uma margem para seu texto.

LI:

A tag li é usada para criar listas, caso você escreva alguma coisa dentro da li, ela vai mostrar no site com uma bolinha atrás do texto.

BR:

A tag br é usada para "pular uma linha" no site.

I:

A tag i é usada colocar o texto em itálico.

B:

A tag b é usada pra colocar o texto em negrito.

IMG SRC:

A tag img src é usada para colocar imagens no site.

MARK:

A tag mark é usada para destacar o texto, Exemplo: EXEMPLO

Q:

A tag q é usada para criar uma citação, colocando o texto entre aspas.

LINK:

A tag link, é usada para linkar imagens, ou arquivos, de outro site para dentro do seu site.
